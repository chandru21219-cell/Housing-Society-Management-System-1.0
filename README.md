# 🏢 Housing Society Management System

A React.js-based Housing Society Management System that helps manage residents, maintenance billing, payments, parking, complaints, and society reports. The application uses **React Context API** and **Local Storage** for state management and data persistence.

---

## 🚀 Features

### 🔐 Authentication

* Login and Logout
* Protected Routes
* Session persistence using Local Storage

### 📊 Dashboard

* Total Residents
* Total Flats
* Total Maintenance Collection
* Pending Collection
* Total Bills
* Paid Bills
* Parking Statistics

### 👥 Resident Management

* Add Resident
* Edit Resident
* Delete Resident
* Search Residents
* Duplicate Flat Validation

### 💰 Maintenance Billing

* Generate Monthly Bills
* Bill Number Generation
* Due Date Tracking
* Paid & Pending Status

### 💳 Payment Management

* Record Maintenance Payments
* Payment Date
* Payment Mode
* Payment History

### 🚗 Parking Management

* Add Parking Slots
* Assign Resident
* Vehicle Details
* Parking Fee
* Occupied/Available Status

### 📝 Complaint Management

* Register Complaints
* Complaint Categories
* Complaint Status
* Delete Complaints

### 📈 Reports

* Resident Summary
* Bill Summary
* Collection Summary
* Pending Collection Report

### ⚙️ Settings

* Application Settings (Extendable)

## ▶️ Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Start the application:

```bash
npm start
```

The application will run at:

```
http://localhost:3000
```

---

## 📌 Current Scope

The current version is a **frontend implementation** with data stored in the browser using Local Storage.
